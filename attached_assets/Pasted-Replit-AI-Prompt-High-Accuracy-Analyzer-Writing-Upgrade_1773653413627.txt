Writing

Upgrade the Ingredient Health Analyzer to use a structured ingredient health database and produce highly accurate food health scoring.

The analyzer must rely on a local ingredient database rather than generic scoring so that each ingredient has a defined category, health impact, and score value.

The system should first parse the ingredient list from user input. Ingredients should be separated primarily by commas, but the parser must also remove brackets, percentages, and extra label text.

Example input:
Sugar, Palm Oil, Cocoa Powder, Flavor Enhancer (MSG), Salt

Normalized ingredients should become:
sugar
palm oil
cocoa powder
msg
salt

Duplicate ingredients must be removed so each ingredient is analyzed once.

Next, connect the parser to a structured ingredient health database. Each ingredient must contain:

ingredient name
category (sweetener, oil, grain, additive, etc.)
health impact level (healthy, moderate, harmful)
base score value
short explanation

Example database entry:

{
"name": "sugar",
"category": "sweetener",
"impact": "harmful",
"score": -20,
"description": "High glycemic sweetener linked to obesity and diabetes risk"
}

Next, implement quantity weighting based on ingredient order.

Food labels list ingredients by quantity, so earlier ingredients should influence the score more strongly.

Weight model:

1st ingredient → weight 1.5
2nd ingredient → weight 1.3
3rd ingredient → weight 1.1
4th ingredient → weight 1.0
5th ingredient → weight 0.8
Remaining ingredients → weight 0.6

The final score should start from a base value of 60 and adjust based on ingredient scores multiplied by their weights.

Clamp the final score between 0 and 100.

Add combination warnings such as:

Sugar + palm oil → ultra processed food warning
Sugar + maltodextrin → high glycemic load warning
Artificial flavor + artificial color → additive-heavy product warning

Limit unknown ingredients to a maximum of five displayed items.

Improve final verdict interpretation:

80–95 → Excellent / Very Healthy
60–79 → Healthy
40–59 → Moderate / Processed
20–39 → Unhealthy
0–19 → Highly Unhealthy / Avoid

The final report must show:

Total detected ingredients
Healthy ingredient count
Moderate ingredient count
Harmful ingredient count
Positive ingredients
Warnings
Overall health score
Final verdict
