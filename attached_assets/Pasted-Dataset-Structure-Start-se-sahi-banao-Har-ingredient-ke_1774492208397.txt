Dataset Structure (Start se sahi banao)

Har ingredient ke liye fixed fields rakho.

Example entry:

{
"id": 1021,
"name": "sugar",
"aliases": ["table sugar","cane sugar","sucrose"],
"category": "sweetener",
"impact": "harmful",
"base_score": -20,
"glycemic_index": "high",
"description": "High glycemic sweetener linked to obesity and diabetes risk"
}

Important fields:

id
name
aliases
category
impact
base_score
description

Aliases future normalization engine me kaam ayenge.

↓

↓
database lookup

60

Quantity multiplier:
